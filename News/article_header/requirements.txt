Exact Naming: The file must be all lowercase: requirements.txt. If you name it Requirements.txt (capital R), Streamlit will not see it.

No Extra Imports: Do not include things like os, time, or ssl in this file. These are built into Python and adding them will break the installation process.

File Location: This file must be in the same folder as your app.py. If your code is in /article_header/app.py, the requirements file must also be inside /article_header/.
